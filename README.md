# Analysis code for [Senescence and individual variation drive egg size and fertility of captive Gymnogyps californianus (California Condors)]  


Files:  

- FinalEggWeight: runs the analysis for egg weight  
- FinalEggFertility: runs the analysis for egg fertility  
- Data/ch1finaldata.csv: data used in the manuscript  

To reproduce results:  

1. Open FinalEggWeight or FinalEggFertility  
2. Set working directory to project root  
3. Run entire script  

R version: 4.3.2  

Key packages: brms, bayesplot, dplyr
